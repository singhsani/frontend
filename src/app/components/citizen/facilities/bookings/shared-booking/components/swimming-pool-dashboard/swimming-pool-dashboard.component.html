વડોદરા મહાનગરપાલિકા હસ્તકના સ્વિમિંગ પૂલની પ્રવેશ ફી અને પ્રવેશ અંગેના નિયમો

  1. મહાનગરપાલિકાના સ્વિમિંગ પૂલના સભ્ય થવા માટે ઠરાવેલ નમૂનામાં અરજી કરવાની રહેશે. આ પ્રવેશપત્ર સ્વિમિંગ પૂલ કચેરીએથી મળી શકશે, તેની કિંમત રૂ.20/-

  2. સભ્યપદ

    1. સ્વિમિંગ પૂલના સભ્ય થવા ઈચ્છતી વ્યક્તિની ઉંમર ૧૮ વર્ષથી ઉપર હોવી જોઈએ.

    2. ૧૮ વર્ષની નીચેના અને ૧૦ વર્ષની ઉપરની વ્યકિત સ્વિમિંગ પૂલના સભ્ય થવા ઈચ્છા ધરાવે તો તેમને તેમના પિતા / વાલીનું સંમતિ અને વિનંતીનું પ્રવેશપત્રમાં દર્શાવ્યા પ્રમાણે સંમતિપત્ર રજૂ કરવાનું રહેશે.

    3. દશ વર્ષની નીચેના બાળકો તેમના પિતા છે વાલી સાથે સહસભ્ય થઈ શકશે. તેમની ફી સામાન્ય ફી જેટલી રહેશે. સહ સભ્યની સંપૂર્ણ જવાબદારી માતા-પિતા કે વાલીની રહેશે. પિતા વાલી તરવૈયા હોવા જરૂરી છે.

    4. શાળાઓ તથા યુનિવર્સિટીમાં ભણતા વિદ્યાર્થીઓ શાળા તથા યુનિવર્સિટીમાંથી મુખ્ય આચાર્ય તથા ડીનનું પ્રમાણપત્ર લાવશે તેમને વિદ્યાર્થી સભ્ય તરીકે દાખલ કરવામાં આવશે.

    5. સ્વિમિંગ પૂલના સભ્ય થવા ઈચ્છતા વ્યકિતની ઓળખાણ નીચે પ્રમાણે આપી શકાશે.

      1. મહાનગરપાલિકાના હાલના મ્યુનિસિપલ સભાસદ

      2. મહાનગરપાલિકાના અધિકારી (વર્ગ-૨) જેમનો પગાર રૂ ૫૩,૧૦૦-૧,૬૭,૮૦૦/- થી ઓછો ન હોય (ભથ્થા સિવાયનો)

      3. સ્કૂલના આચાર્ય અને કોલેજના ડીન

      4. ગેઝેટેડ સરકારી અધિકારી

    6. બાળ સ્વિમિંગ પૂલના વધારાના નિયમો :

      1. બાળ સ્વિમિંગ પૂલમાં ૫ થી ૧૨ વર્ષની ઉંમરના બાળકો પ્રવેશપાત્ર રહેશે.

      2. બાળ સ્વિમિંગ પૂલ બેંચનો સમય ૪૫ મીનીટનો જ રહેશે. સમય સવારે ૭-૦૦ થી ૧૦- ૦૦ તથા સાંજે ૪-૦૦ થી ૭-૦૦ રહેશે.

      3. બાળ સ્વિમિંગ પૂલના બાળ સભ્યોએ ફલોટીંગ એપેટર્સ જેમ કે ફલોટીંગ રીંગ,આર્મબેન્ડ, ફ્લોટર્સ વિગેરે સાધનો સભ્યોએ લાવવાના રહેશે.

  3. મહાનગરપાલિકા સ્ટાફ તથા મ્યુનિસિપલ પેન્શનર તથા માજી/ચાલુ સભાસદશ્રીઓ : શાળા તથા કોલેજમાં અભ્યાસ કરતાં વિદ્યાર્થીઓને તેમની શાળા કે કોલેજોનાં મુખ્ય આચાર્યનું સર્ટીફીકેટ રજૂ કરવું પડશે. તેની સભ્ય ફી નીચે પ્રમાણે રહેશે (ચાલુ વર્ષનું બોનાફાઈડ સર્ટીફીકેટ વિદ્યાર્થીઓને લાવવાનું રહેશે.)

  4. વડોદરા મહાનગરપાલિકાના સ્ટાફના કાયમી કર્મચારીઓનું સભ્યપદ વડોદરા મહાનગરપાલિકાના સ્ટાફના કાયમી કર્મચારી જો સ્વિમિંગ પૂલના સભ્ય થવું હશે તો અલગ બતાવ્યા પ્રમાણેનું ફીનું ધોરણ રહેશે. આ માટે જે તે કર્મચારીએ ખાતાધિકારીનો દાખલો રજુ કરવાનો રહેશે.

  5. માજી/ચાલુ મ્યુનિસિપલ સભાસદશ્રીઓ મ્યુનિસિપલ સેક્રેટરીશ્રીનું પ્રમાણપત્ર રજૂ કરવાનું રહેશે. :

  6. તરણ સ્પર્ધા: માન્ય સંસ્થાઓએ જે તે સ્વિમિંગ પૂલમાં સ્પર્ધા યોજવા માટેની અઠવાડીયા અગાઉ પરવાનગી મેળવી લેવાની રહેશે.તરણ સ્પર્ધાનાં આયોજનના દિવસના કે તેના ભાગ માટે ૨ ૧૦,૦૦૦/- ફી ભરીને તરણ સ્પર્ધા યોજી શકશે તથા ગપુલનો ઉપયોગ તરણ સ્પર્ધાના સમય પૂરતો જ મર્યાદિત રહેશે.

નોંધ : જે સભ્યો શીખાઉ હશે વ્યકિત દીઠ રૂ ૩૦૦/- શીખાઉ ફી માસિક ધોરણે વ્યકિતગત વધારાની ભરવાની રહેશે.

નિયમ અને સૂચનાઓ :

  1. સ્વિમિંગ પૂલનું વર્ષ એપ્રિલથી માર્ચ સુધીનું ગણવામાં આવશે. (દરેક નવા નાણાંકીય વર્ષ ફોર્મ ભરી પ્રવેશ મેળવવોરહેશે. ફી માસવારી રહેશે.)

  2. સ્વિમિંગ પૂલના સભ્ય થવા ઈચ્છા ધરાવનાર દરેક અરજદારે મહાનગરપાલિકા દવાખાનાના ડૉક્ટરો અથવા મા ડૉક્ટરો એમ. બી. બી. એસ. કે તેનાથી વધુ કવોલીફીકેશનવાળા ડૉક્ટરોનું પ્રમાણપત્ર પ્રવેશપત્રમાં દર્શાવ્યા પ્રમાણે મેળવવાનું રહેશે.

  3. સ્વિમિંગ પૂલનો લાભ લેનાર વ્યક્તિએ તરતી વખતે તરવાનો ખાસ રંગીન પોષાક પહેરવાનો ફરજીયાત રહેશે. બહેનોએ તેમજ લાંબાવાળ ધરાવનાર વ્યક્તિએ ટોપી પહેરવાની ફરજીયાત રહેશે.

  4. તરવૈયા તરીકેનો સીધો પ્રવેશ મેળવવા માટે જે તે વ્યક્તિને ઓછામાં ઓછુ ૫૦ મીટર તરતાં આવડવું જોઈએ. આ માટે તેઓએ રૂ ૩૦/- ભરીને ટેસ્ટ આપવાનો રહેશે. આ અંગેનો ટેસ્ટ જે તે સ્વિમિંગ પૂલના માનદ્દ ટ્રેનર્સ/કોચને આપવાનો રહેશે. સદર ટેસ્ટમાં ઉત્તીર્ણ થયા બાદ તેઓએ સીધા તરવૈયા તરીકે પ્રવેશ મળી શકશે.

  5. સ્વિમિંગ પૂલમાં દાખલ થનાર સભ્ય નિયત કરેલા સ્થળે સ્નાન કર્યા બાદ જ સ્વિમિંગ પૂલમાં પ્રવેશ કરવાનો રહેશે. સ્વિમિંગ પૂલમાં દાખલ થતી વખતે તેલ, સાબુ, કે એવા કોઈ પદાર્થ સભ્ય શરીર પર લગાવેલો હોવો જોઈએ નહીં.

  6. સ્વિમિંગ પૂલનો કોઈપણ સભ્ય સ્વિમિંગ પૂલમાં કે તેના કોઈપણ અન્ય સ્થળે કોઈપણ પ્રકારની ગંદકી કે ધુમ્રપાન કરશે નહી. અને સ્વિમિંગ પૂલના અધિકારીની સ્વચ્છતા અંગેની સૂચનાનું પાલન કરશે. સ્વિમિંગ પૂલમાં કોઈપણ સભ્ય કૂતરો કે પ્રાણી લાવી શકશે નહીં.

  7. સ્વિમિંગ પૂલના બેચના દરેક સમયે મ્યુનિસિપલ કમિશનરશ્રી અગર ખાતાધિકારી નક્કી કરશે તે મુજબના રહેશે. સ્વિમિંગ પૂલને ખુલ્લું કે બંધ રાખવાનો સમય સ્ત્રી - પુરૂષ કે શિખાઉ માટેના અલગ અલગ સમય વિગેરે મ્યુનિસિપલ કમિશનરનો નિર્ણય આખરી ગણાશે.

  8. કોઈપણ સભ્ય દિવસમાં એક વખત 45 મીનીટ માટે સ્વિમિંગ પૂલનો ઉપયોગ કરી શકશે. સમય પૂરો થયાની સૂચના મળતા બેલ વાગતાં દરેક સભ્ય સ્વિમિંગ પૂલમાંથી બહાર નીકળી જવું ફરજીયાત રહેશે.

  9. સ્વિમિંગ પૂલમાં દાખલ થતાં પહેલાં અને તેમાંથી બહાર નીકળ્યા પછી દરેક સભ્ય સ્વિમિંગ પૂલના રજીસ્ટરમાં ભરવાની વિગત અને સહીં સ્પષ્ટ અક્ષર વડે ભરવી આવશ્યક રહેશે.

  10. લોકર્સ : સ્વિમિંગ પૂલમાં લોકર્સ માટે સભ્ય ત્રિમાસિક રૂ ૧૫૦/- અથવા વાર્ષિક ફી રૂ ૫૦૦/- ભરવાની રહેશે, તેમજ અનામત રૂ ૫૦/- ભરવાના રહેશે.

  11. સ્વિમિંગ પૂલના ડ્રાઈવીંગ બોર્ડ અથવા અન્ય કોઈ મિલ્કતનો સભ્ય કોઈ દુરઉપયોગ કરશે અથવા અશિસ્તભર્યું (ગેરવર્તણુંક) વર્તન કરશે અથવા સ્વિમિંગ પૂલના કોઇપણ નિયમનો ભંગ કરશે તો તેને બહાર કાઢવા સ્વિમિંગ પૂલના અધિકૃત અધિકારીની સંપૂર્ણ સત્તા રહેશે. આવા સભ્યપદ રદ્દ કરવા અંગે મ્યુનિસિપલ કમિશનરશ્રીને સંપૂર્ણ સત્તા રહેશે, તેમજ સ્વિમિંગ પૂલને થયેલ નુકશાન આવા સભ્ય પાસેથી વસૂલ કરવાની મ્યુનિસિપલ કમિશનરશ્રીને સત્તા રહેશે.

  12. સ્વિમિંગ પૂલ ના નિયમો અથવા સ્વિમિંગ પૂલને સ્પર્શતી કોઈપણ બાબત વિષે ઉપસ્થિત થયેલ પ્રશ્ન અંગે મ્યુનિસિપલ કમિશનરશ્રીનો નિર્ણય આખરી ગણાશે.

  13. સ્વિમિંગ કોસ્ચુમ વગર કોઈપણ સભ્યને પ્રવેશ આપવામાં આવશે નહી. સ્વિમિંગ કોસ્ચુમ દરવાજે ચેકરને બતાવવાના રહેશે.

  14. તોફાની સભ્યનું સભ્યપદ તેમની વર્તણૂક સંતોષકારક નહીં લાગે તો સ્વિમિંગ પૂલ બહાર કાઢવામાં આવશે. અને સભ્યપદ રદ કરવામાં આવશે જેમાં તકરાર કે અપીલ ચાલશે નહીં.

  15. પ્રવેશ ફોર્મ સાથે બે ફોટા લાવવાના રહેશે.

  16. માન્ય સંસ્થાઓ તરફથી સ્પર્ધા યોજે તે દિવસે અને ગુરૂવારે સ્વિમિંગ પૂલ બંધ રહેશે.

  17. સભ્યએ જે સ્નાનાગારમાં પ્રવેશ મેળવેલ હશે તેનો પ્રવેશ તે સ્નાનાગાર પૂરતો મર્યાદિત રહેશે.

  18. દરેક સભ્યએ પોતાનું કાર્ડ પ્રવેશ કરતાં પહેલાં કાર્ડ ચેકરને અવશ્ય બતાવવાનું રહેશે અને ત્યારબાદ જ સ્વિમિંગ પૂલમાં પ્રવેશન પાત્ર રહેશે તથા સ્વિમિંગ પૂલના કોઈપણ સભ્યનું કાર્ડ ખોવાઈ જાય તો રૂ ૧૦૦/- લાગત ફી લઈ ડુપ્લીકેટ કાર્ડ કાઢી આપવામાં આવશે. ડુપ્લીકેટ કાર્ડ માટે સભ્યે પોતાનો ફોટો નંગ-૧ આપવાનો રહેશે.

  19. સ્વિમિંગ પૂલમાં કોઈ સભ્ય કે મહેમાનને અકસ્માત, શારીરિક હાની વ્યથા કે મૃત્યુ થાય કોઈ ચીજ વસ્તુ ખોવાય તે અંગેની કોઈપણ જવાબદારી મહાનગરપાલિકાની અથવા મ્યુનિસિપલ કમિશનર અગર આ અંગેની સત્તા આપી હોય તેવા અધિકારીની રહેશે નહીં.
